OPERATING SYSTEM (OS)  DAY 1 20/02/26

A bridge between user/applications and hardware [CPU AND RAM...].

CPU → Executes instructions loaded in RAM by OS
RAM → Required for all CPU processing

------------------------------------------------------------------------------------------------------------+

Windows Storage Drives (Historical Examples):

A: & B: Drives → Reserved for floppy disks

C: Drive → OS and applications

D:, E:, F:, etc. → Additional storage drives

Windows versions: 95, 98, XP, 2007, 2008, 2010, 2011 (current & still developing)

------------------------------------------------------------------------------------------------------------+

PROGRAM & PROCESS:

Program: Any file in storage . Passive entity.


Process: Program under execution. Active entity (uses CPU time + RAM allocation)

------------------------------------------------------------------------------------------------------------+

OS Features:

Process Management (CPU scheduling)

Memory Management (RAM allocation)

I/O Management

Secondary Storage Management

Security (isolates programs/data - no sharing between concurrent processes)

------------------------------------------------------------------------------------------------------------+

Memory Types:

RAM (Random Access Memory) → Volatile, read/write


ROM (Read Only Memory) → Non-volatile, read-only

------------------------------------------------------------------------------------------------------------+

DATA MEASUREMENT UNITS

1 Bit = 0 or 1 (smallest unit)
1 Byte = 8 Bits = 1 character

1 KB = 1024 Bytes
1 MB = 1024 KB = 1,048,576 Bytes
1 GB = 1024 MB = 1,073,741,824 Bytes  
1 TB = 1024 GB = 1,099,511,627,776 Bytes

-----------------------------------------------------------------------------------------------------------+
